## Example Code

```tsx
<Container/>
```

## Usage Guideline

| Prop       | Usage                                                                                                                                                             |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| background | This prop should be equal to "light-gray" in order to add a class to the container that will provied the background color. Now it only support light-gray ($N100) |
| fixed      | When it is true, it adds a class to the container to make it position fixed with 99 z-index                                                                       |
| flexbox    | When it is true, it adds a class to the container to make it display flex                                                                                         |
