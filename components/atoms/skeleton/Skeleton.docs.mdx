## Usage guideline

- If only a portion of a page is displaying new content or being updated, place the Skeleton in that part of the page.
- If you are unsure where to place the Skeleton, place it where you want the user's attention to be when loading is finished.
- Only show a Skeleton if the expected wait time is more than a second.
