### Usage guidelines

- The entire badge is clickable/tappable.
- “Initial” Avatar should be default when photo is not available.
- Contact menu is a [Dropdown](https://ux-framework-svc-sandbox-us.app.pplsicorp.com/?path=/docs/controls-dropdown--links-dropdown) component and should follow the same rules.
- Associate name or title should be single line. Truncate it with “…” when necessary.
