## Usage

If you need to use some font in a custom component, these are also being exported as css variables so they can be used like the following example once you have the dist/index.css file imported in your project.

```css
.some-class {
  font: var(--t14); // $T14
}
```

Also you can use these as a React component using the **`Heading`** component

  <Heading as="T28" text="Heading Example" />
</Canvas>
