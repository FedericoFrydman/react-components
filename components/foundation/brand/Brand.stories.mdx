# Brand

Customization is available based on the brand allowing for differentiation. Brand data is set into css variables at runtime. Elements of the UX framework uses these variables.

# Typography

The brand font is availalbe as var(--brand-font-family). Its use is limited to the key marque areas. As such, the &lt;Heading&gr; component allows for the useBrand attribute.

Fonts in the heading when the brand is used will have different sizes, as the intent is that it is for calling out brand data.

For LegalShield this is the font "Iner". This is a serif font.
