# Spacing

Spacings help build visual hierarchy, create rhythm for users to digest content at ease.
Olympus uses an 8-point grid system. A layout or component padding, margin, height, etc. should always be a multiple of 8 pixels. In consideration for limited real estate, half point values (2px, 4px) are also available.
The values below are for horizontal and vertical spacing.

**In order to use these classes you have to import the `utilities.scss`file into component file.**

## Values

Pixel values are based on a font-size of `16px` which gives an equivalent of `1rem` unit to `16px`.

| Scale | PX Value | REM Value |
| ----- | -------- | --------- |
| 1     | 2px      | 0.125rem  |
| 2     | 4px      | 0.25rem   |
| 3     | 8px      | 0.5rem    |
| 4     | 16px     | 1rem      |
| 5     | 24px     | 1.5rem    |
| 6     | 32px     | 2rem      |
| 7     | 40px     | 2.5rem    |
| 8     | 48px     | 3rem      |
| 9     | 56px     | 3.5rem    |
| 10    | 64px     | 4rem      |
| 11    | 72px     | 4.5rem    |
| 12    | 80px     | 5rem      |

## How it works

Assign margin or padding values to an component or a subset of its sides with shorthand classes.
Includes support for individual properties, all properties, and vertical and horizontal properties.
Classes are built from a default Sass map ranging from .125rem to 5rem.

## Notation

The classes are named using the format `{property}{sides}-{size}`

**Where property is one of:**

- **m** &#8594 for classes that set margin
- **p** &#8594 for classes that set padding

**Where sides is one of:**

- **t** &#8594 for classes that set margin-top or padding-top
- **b** &#8594 for classes that set margin-bottom or padding-bottom
- **l** &#8594 for classes that set margin-left or padding-left
- **r** &#8594 for classes that set margin-right or padding-right
- **x** &#8594 for classes that set both _-left and _-right
- **y** &#8594 for classes that set both _-top and _-bottom
- **\_blank** &#8594 for classes that set a margin or padding on all 4 sides of the element\_

**Where size is one of:**

- **0** &#8594 for classes that eliminate the margin or padding by setting it to 0
- **1** &#8594 (by default) for classes that set the margin or padding to $spacer \* .125
- **2** &#8594 (by default) for classes that set the margin or padding to $spacer \* .25
- **3** &#8594 (by default) for classes that set the margin or padding to $spacer \* .5
- **4** &#8594 (by default) for classes that set the margin or padding to $spacer
- **5** &#8594 (by default) for classes that set the margin or padding to $spacer \* 1.5
- **6** &#8594 (by default) for classes that set the margin or padding to $spacer \* 2
- **7** &#8594 (by default) for classes that set the margin or padding to $spacer \* 2.5
- **8** &#8594 (by default) for classes that set the margin or padding to $spacer \* 3
- **9** &#8594 (by default) for classes that set the margin or padding to $spacer \* 3.5
- **10** &#8594 (by default) for classes that set the margin or padding to $spacer \* 4
- **11** &#8594 (by default) for classes that set the margin or padding to $spacer \* 4.5
- **12** &#8594 (by default) for classes that set the margin or padding to $spacer \* 5
- **\_auto** &#8594 for classes that set the margin to auto\_

## Examples

Here are some representative examples of these classes:

<Source
  language="css"
  code={`
    .mt-0 {
      margin-top: 0;
    }
    .ml-1 {
      // ($spacer * .125)
      margin-left: 0.125rem;
    }
    .px-2 {
      // ($spacer * .25)
      padding-left: 0.25rem;
      padding-right: 0.25rem;
    }
    .p-3 {
      // ($spacer = 1rem)
      padding: 1rem;
    }
  `}
/>
