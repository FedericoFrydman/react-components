# Select Props

<Description />

**This documentation is just to display the select props where you can interact with them and see how it behaves. For implementation please use [FormField](/?path=/docs/forms-form-fields--select-field) docs to gide.**

<Alert title="Dev Note" severity="warning" icon="alert_warning">
  The Select component must be used as a child of a FormField component.

### Select Placeholder

In order to have a placeholder in your select component, you need to set the `placeholder` prop.

### Label

Pass a `label` prop to the `FormField` component for a11y reasons.

## Example Code

```tsx
<FormField id="select-formfield" label="Some label:">
  <Select>
    <SelectItem textValue="USA" value="usa" />
    <SelectItem textValue="Mexico" value="mexico" />
    <SelectItem textValue="Canada" value="canada" />
  </Select>
</FormField>
```
