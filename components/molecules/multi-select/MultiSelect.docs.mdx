### MultiSelect Placeholder

In order to have a placeholder in your select component, you need to set the placeholder and placeholerValue prop. The defaultValue on select will take the placeholerValue and make it appear on the select component.

### Label

Pass a `label` prop to the `FormField` component for a11y reasons.

## Example Code

```tsx
<FormField id="select-formfield" label="Some label:">
  <MultiSelect
    options={[
      {
        label: 'United States',
        value: 'US',
      },
      {
        label: 'Germany',
        value: 'DE',
      },
    ]}
  />
</FormField>
```
