# Badge

A badge is a visual element to quickly indicate the status, state, or category of a piece of information. Users cannot interact with badges. If the interaction is desired, use a Tag.

## Usage guideline

- **Do** write label as a single keyword, short and precise. Use two words only if necessary.
- **Do not** include an icon inside a badge. The badge is a text label only.
- Labels should remain case sensitive.
- Keep label text precise.
