# Checkbox

Checkbox allows multiple selections from a set of options. User can select none, one or multiple.
Checkbox component is always paired with a label. It can be single, as a group, or nested.
Checkbox recieves an onClick prop in order to handle preventDefault or stopPropagation functions. The onClick prop is not required, and it should work without it.

**This documentantion is just to display the input props where you can interact with them and see how it behaves. For implementation please use [FormField](/?path=/docs/forms-form-fields--checkbox-field) docs to gide.**

### Nested Checkboxes

Checkboxes may be nested to show a hierarchy of selections. Selecting a parent will automatically activate and select all children.

Deselecting a parent checkbox will deactivate and deselect all children. If a group of child checkboxes contains a mixture of checked and unchecked, show an indeterminate state with a dash.

## Guidelines

<p style={{ margin: 0 }}>&nbsp;</p>

- **Do** Always provide a label.
- **Do** Write labels in sentence case.
- **Do not** Use punctuation in labels or after checkbox fields.
- **Do** Keep labels short so users can select quickly and confidently.
- **Do** Write labels as statements that the selected state makes true and the unselected state makes false.
- **Do** Title a checkbox group if the content of a group of checkboxes is not immediately apparent, provide a title to clarify.
- **Do** Both the checkbox and label should trigger selection.
